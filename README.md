## 📖 About

CppVerse is a collection of C++ projects developed to strengthen programming fundamentals, object-oriented design, problem-solving skills, and software development practices.

This repository showcases projects ranging from management systems to interactive games, demonstrating practical applications of C++.

---

# 📂 Projects Included

## 🏦 Bank Management System

A console-based banking application that simulates basic banking operations.

### Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Account Information Management
- Menu-Driven Interface

### Concepts Used

- Classes & Objects
- File Handling
- Functions
- Data Management
- Object-Oriented Programming

---

## 🚗 Car Game

An interactive car racing game developed using C++.

### Features

- Real-time Controls
- Obstacle Avoidance
- Score Tracking
- Collision Detection
- Dynamic Gameplay

### Concepts Used

- Game Development
- Loops & Conditions
- Keyboard Input Handling
- Real-Time Processing

---

## 🎮 Moving C

A graphics-based animation project featuring a moving object represented by the letter "C".

### Features

- Object Animation
- Movement Controls
- Graphics Rendering
- Interactive Experience

### Concepts Used

- Graphics Programming
- Animation Logic
- Coordinate Systems
- Event Handling
